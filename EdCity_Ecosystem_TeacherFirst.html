EdCity.aiThe teacher’s daily-use core
Prep & materials
  • Prepares for class, generates and validates teaching materials against the official curriculum framework
  • Collaborates cross-subject — invites colleagues from other subjects to co-review materials
Assignments & marking
  • Generates assignments using a mechanism that encourages students to think before receiving AI help
  • Marks assignments with the same thinking-first approach, including open-ended written work
Groups & communication
  • Creates and manages teaching groups; assigns materials by group
  • Drafts admin / home-school communications
  • Shares resources; browses and uses partner (vendor) tools via the tool library
Insight
  • Views class-level skill profile by learning tier (stretch / core / support), including causal before-after insight
Subject-lead — acting on the teacher’s behalf
  • Reviews marking calibration and sets department-wide policy
